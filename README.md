都享券 (Duxiang Quan) - 數字化運營與財務駕駛艙

基於《都享券項目申請報告 V2》設計的項目執行與資金營收演示系統。本項目旨在通過數字化儀表盤，向投資者與管理層展示項目的運營監控邏輯與商業價值模型。

🚀 項目簡介

都享券 是一個基於數字聯券技術的新消費項目，旨在通過「集採讓利 + 免費派券」的模式，連接餐飲商家與消費者，打造城市消費一券通。

本演示系統（Demo）包含兩個核心視圖：

項目執行監控 (Operational Dashboard)：實時展示商家接入情況、用戶增長、交易熱力圖及風控預警體系。

資金與營收模型 (Financial Model)：可視化展示 10 億投資分期計劃、資金周轉率預測及 3 年 ROI（投資回報率）測算。

🛠️ 技術棧

本項目為純靜態單頁應用（SPA），無需後端即可運行：

HTML5 / CSS3

Bootstrap 5 (響應式佈局與 UI 組件)

Chart.js (動態圖表繪製)

FontAwesome (圖標庫)

JavaScript (ES6) (模擬數據邏輯與交互)

📊 功能演示

1. 運營監控端

核心指標：合作商家數、註冊用戶數、實時 GMV。

地圖模擬：廣州市商家分佈與實時交易脈衝動畫。

風控看板：模擬「准入-監測-追償」體系，展示異常商家預警邏輯。

2. 財務預測端

投資計劃：一期/二期/三期資金投入佔比（餅圖）。

ROI 測算：基於 3% - 5% 淨利率模型的三年利潤預測（柱狀圖+折線圖）。

動態表格：點擊按鈕可切換查看不同年份的詳細財務數據（流水、周轉率、壞賬率）。
